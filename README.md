# Taffy High Contrast Theme for Helix Editor

## Screenshots

![](images/code.png)

## Quick Start

```bash
git clone https://github.com/kalifun/taffy.helix.git

mv themes ~/.config/helix/
```

You need to modify `~/.config/helix/config.toml`:

``` toml
theme = "taffy16"
```
